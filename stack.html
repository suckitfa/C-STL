C++ STL 中的 stack基本使用方法:

stack在头文件stack中，所以使用时应该加上 #include

申明一个栈 stack<数据类型> 栈的名字;

//举例说明：
stack<int> s;
stack<string> s;

常用方法：

  • 加入数据 push( 待加入的数据 );
  • 求元素个数 size()
  • 访问栈顶元素 top()
  • 删除栈顶元素 pop();
  • 是否为空 empty(); 返回值 空：0 非空 ：1
xxxxxxxxxx
#include <iostream>
#include <stack>
using namespace std;
int main(){
  stack<int> stk;
  //入栈6个元素
   stk.push(1);
   stk.push(2);
   stk.push(3);
   stk.push(4);
   stk.push(5);
   stk.push(6);
  
  cout<<"Is the stack empty? = " << stk.empty()<<endl;//0==false
  cout<<"The number of elments in the stack : = "<<stk.size()<<endl;
  while(!stk.empty()){
        // 访问栈顶的元素
        cout<<stk.top()<<" ";
        //删除栈顶的元素
        stk.pop();
    }
    cout<<endl;
  
  return 0;
}

栈作为一种线性表，限制它在栈顶存储元素,因此搞清楚哪里是栈顶很重要！！！

加入的新元素自动成为栈的顶部！！！
